Holepunch package
=================

Holepunch package provides a server and client to enable P2P communication
accross NAT/Firewall. A Client behind a NAT/Firewall will be introduced to
another by the Server. Each client will then send UDP/TCP packets to the
destination client in order to punch a hole and enable P2P communication.
